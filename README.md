# Elevence Tasks - Internship Project

## 📋 Project Overview

**Elevence Tasks** is a comprehensive data visualization and analytics project completed during my internship. This repository showcases an in-depth analysis of e-commerce metrics, focusing on pricing strategies, promotional effectiveness, discount impact, financial risk assessment, trust-weighted ratings, and context-aware price banding.

---

## 🎯 Project Objectives

- Analyze relationships between pricing and customer satisfaction
- Evaluate promotional campaign effectiveness
- Assess discount impact on product ratings
- Identify and quantify financial risk factors
- Develop trust-weighted rating systems
- Implement context-aware price banding strategies
